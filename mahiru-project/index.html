Apa itu Mahiru Project?

Mahiru Project adalah project yang dikembangkan oleh saya sendiri untuk menghasilkan output sebuah minecraft server bergenre Anarchy.

  • Project ini bermula dari keinginan saya untuk mempopulerkan kembali genre Anarchy di komunitas minecraft. Sebuah genre dimana kebebasan adalah kunci utama dari game minecraft. Tanpa peduli cheater ataupun bukan, semuanya diperbolehkan.
  • Namun, dari sekian banyak ambisi yang saya punya, saya terhalang satu kondisi dimana kurang diminatinya genre Anarchy dan mulai menipisnya budget untuk menutup biaya operasional server. Membuat saya terpaksa harus menutup Server karena biaya yang semakin membengkak.
  • Mahiru Survival memiliki map yang tidak terlalu besar ukurannya mengingat usia server hanya 8 Bulan. Kamu bisa mendownload Map server ini dengan tombol dibawah.
